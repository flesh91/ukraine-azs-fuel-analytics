# ukraine-azs-fuel-analytics
SPA-дашборд на Node.js та Vanilla JS для аналізу ринку пального в Україні. Автоматично збирає ціни Мінфіну та курси валют НБУ, розраховує математичну модель справедливої вартості палива на основі котирувань нафти Brent та податкового законодавства. Підсвічує аномальні відхилення на графіку Chart.js та підтримує гнучке кешування даних.
